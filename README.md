# PhysicsLab AI 光学仿真

最新版 React + TypeScript 仿真工程。

## 在线 Demo

GitHub Pages 部署地址：

```text
https://luyi210.github.io/physics-lab-ai/
```

说明：

- 基础助教可直接在线体验。
- 本地模型增强只会连接访问者自己电脑上的 Ollama，不会连接开发者电脑。

## 运行

```bash
npm install
npm run dev
```

打开：

```text
http://127.0.0.1:5173/
```

## 构建

```bash
npm run build
```

## 当前模块

- 光的折射
- 全反射
- 介质折射率调节
- 入射角拖动与自动扫角
- 实时折射角、临界角、反射强度、光速比例读数
